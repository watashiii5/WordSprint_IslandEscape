# Word Sprint: Island Escape

A side-scrolling educational word game where you help a character escape an island by answering vocabulary questions to jump across gaps.

## How to Play

- Your character automatically walks forward across floating platforms
- When you reach a gap, a vocabulary question appears with 3 answer choices
- Click the correct answer to jump across — get it wrong and you'll see a hint before trying again
- Answer all 5 questions to escape the island and see your score

## Features

- Procedurally generated character with walk, idle, and jump animations
- Parallax scrolling backgrounds with clouds, mountains, and sun
- Animated water with wave effects under platform gaps
- Bouncy UI animations for questions and answer buttons
- Score tracking with sparkle effects on correct answers
- Hint system that provides feedback on wrong answers
- Confetti celebration on the success screen

## Tech Stack

- **Game Engine:** Phaser 3.80.1
- **Language:** Vanilla JavaScript
- **Rendering:** HTML5 Canvas
- **Bundler:** Vite
- **UI Framework:** React + TypeScript (scaffold, not used by game)
- **Styling:** Tailwind CSS + Radix UI (scaffold, not used by game)
- **Package Manager:** pnpm
- **Hosting:** Replit / Vercel
